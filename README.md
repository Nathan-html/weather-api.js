# About

weather-api.js is a simple Node.js module that allows you to easily interact with the API of weatherapi.com.

# Installation

```cmd
npm install @luaft/weather-api.js
yarn add @luaft/weather-api.js
pnpm add @luaft/weather-api.js
```
